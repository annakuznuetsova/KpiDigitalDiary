# KpiDigitalDiary
DigitalDiary — це вебзастосунок, призначений для ведення персонального цифрового щоденника та керування користувацьким контентом. Система дозволяє користувачам реєструватися, проходити автентифікацію, керувати власними записами та налаштовувати головну сторінку. Проєкт побудований на основі ASP.NET Core та відповідає принципам чистої архітектури з повним покриттям модульними тестами.
